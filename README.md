## Instalação

```bash
# Baixe o repostitório com os agentes.
git clone https://github.com/datamllab/rlcard.git

# Crie uma virtualenv.
python3 -m venv venv

# Inicie a virtualenv.
source venv/bin/activate

# Vá para a pasta do repositório.
cd rlcard

# Execute o setup das dependências.
python setup.py
```
