## Instalação

```bash
# Crie uma virtualenv.
python3 -m venv venv

# Inicie a virtualenv.
source venv/bin/activate

# Instale as dependências.
pip install -r requirements.txt
```

## Sobre o projeto

Nós utilizamos o [rlcard](https://github.com/datamllab/rlcard) para os algoritmos de aprendizagem por reforço com blackjack.
